- View membership status

- Track free service eligibility
- Upgrade options

- Update contact info
- Change notification preferences
- View billing history
- Manage payment methods

## Client FAQs

### "How do I log in?"
Clients access via email link or create account with:
- Email address
- Phone number
- Password

### "I forgot my password"
Click "Forgot Password" ‚Üí Enter email ‚Üí Check inbox for reset link

### "How do I cancel an appointment?"
1. Go to My Appointments
2. Find the appointment
3. Click "Cancel"
4. Confirm cancellation

*Note: Cancellation policy applies (<24 hours may incur fee)*

### "How do I earn rewards?"
- Book and complete appointments

- Leave Google reviews
- Birthday bonus points

### "Is my information secure?"
Yes! We use:
- HIPAA-compliant systems

- Secure data centers
- Regular security audits

## Troubleshooting

### Client Can't Log In
1. Verify email is correct in their profile
2. Have them try "Forgot Password"
3. Check if account is active
4. Clear browser cache/cookies

### App Won't Install
1. Must use Safari (iPhone) or Chrome (Android)
2. Ensure latest browser version
3. Try refreshing the page first
4. Check available storage on device

### Forms Not Submitting
1. Check internet connection
2. Ensure all required fields filled
3. Try different browser
4. Clear cache and retry
