# Daily Operations Checklist

**Hello Gorgeous Med Spa - Daily Operations Manual**  
**Version:** 1.4.0 | **Updated:** January 31, 2026

---

## Morning Opening (8:30 AM)

### System Check
```
□ Open admin dashboard: hellogorgeousmedspa.com/admin
□ Verify database connection (no "Demo Mode" warning)
□ Check today's date displays correctly
```

### Review Dashboard
```
□ Note total appointments for today
□ Check for unsigned charts (complete before EOD)
□ Check for expiring consent forms
□ Review any overnight messages/notifications
```

### Calendar Review
```
□ Open /admin/calendar
□ Verify all appointments showing
□ Note any special requests in appointment notes
□ Identify VIP clients for special attention
□ Check for buffer time between procedures
```

### Inventory Check
```
□ Open /admin/inventory
□ Check low stock alerts
□ Verify injectable lot numbers for today's procedures
□ Check expiration dates on products being used
```

### Provider Prep
```
□ Print or review daily schedule
□ Pull client charts for first appointments
□ Verify consent forms signed for all procedures
□ Prepare treatment rooms
```

---

## During Business Hours

### Client Arrival
```
□ Greet client by name
□ Verify appointment in system
□ Mark as "Checked In" on calendar
□ Confirm consent forms signed (if needed)
□ Collect any outstanding balances
□ Direct to treatment room when ready
```

### During Treatment
```
□ Provider marks status as "In Progress"
□ Document treatment in chart
□ Take before/after photos (if applicable)
□ Record lot numbers for injectables
```

### Post-Treatment
```
□ Mark appointment as "Completed"
□ Process payment (or note if billed later)
□ Book next appointment
□ Provide aftercare instructions
□ Request Google review (for 5-star experiences)
```

### New Client Workflow
```
1. □ Create client profile in /admin/clients/new
2. □ Send intake forms digitally
3. □ Verify ID (save copy in profile)
4. □ Have client sign required consent forms
5. □ Take initial photos
6. □ Complete consultation
7. □ Book first treatment
```

---

## Client Communication

### Appointment Confirmations
- Sent automatically 48 hours before
- Clients can confirm via link

### Day-Before Reminders
- Sent automatically 24 hours before
- Include arrival time and prep instructions

### No-Show Follow-Up
```
□ Mark appointment as "No Show"
□ Send reschedule message (SMS or call)
□ Note in client profile
□ Apply no-show policy if applicable
```

### Cancellation Handling
```
□ Mark appointment as "Cancelled"
□ Note cancellation reason
□ Check cancellation policy (< 24 hours = fee?)
□ Offer to reschedule
□ Update waitlist if applicable
```

## End of Day (Close)

### Financial Reconciliation
```
□ Review all payments processed today
□ Verify totals match appointments completed
□ Note any outstanding balances
□ Check for refunds or adjustments
```

### Chart Completion
```
□ All charts signed by provider
□ Treatment notes completed
□ Photos uploaded
□ Lot numbers recorded
```

### System Tasks
```
□ Review tomorrow's schedule
□ Identify clients needing consent renewal
□ Check inventory for tomorrow's procedures
□ Note any special instructions for opening
```

### Physical Close
```
□ Secure all medications/controlled substances
□ Lock treatment rooms
□ Secure cash/payment terminals
□ Set alarm
```

## Emergency Procedures

### System Down
1. Use paper backup for appointments
2. Process payments manually (Square, etc.)
3. Enter data when system restored
4. Contact support if prolonged

### Medical Emergency
1. Call 911 immediately
2. Follow emergency protocols (see compliance/incident-response.md)
3. Document incident
4. Notify insurance carrier

### Data Breach Suspected
1. Do not access affected systems
2. Document what was observed
3. Contact IT/support immediately
4. Follow incident response plan
